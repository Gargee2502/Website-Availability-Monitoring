# Website Availability Monitoring System
A Linux-based monitoring platform that continuously checks website availability and performance.

## Features
DNS resolution monitoring
Ping connectivity checks
HTTP status monitoring
Response time measurement
SQLite database logging
Telegram alert notifications
Uptime analytics
Web dashboard

## Tech Stack
Bash
Python (Flask)
SQLite

## System Architecture
Monitoring Script (Bash)
        ↓
SQLite Database
        ↓
Alert System (Telegram)
        ↓
Flask API
        ↓
Web Dashboard

## Dashboard
Displays:

Website status (UP/DOWN)
Response time
Monitoring history
Uptime analytics graph
